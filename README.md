# Game-User-Analytics-Dashboard-Tableau
Interactive Tableau dashboard analyzing user engagement, playtime and activity patterns

## 📊 Project Overview
Interactive Tableau dashboard analyzing game user engagement, playtime and activity patterns.

## 🛠️ Tools & Technologies
- **Tableau Public** — interactive dashboard and visualizations
- **Excel / CSV** — data preparation

## 📈 Key Metrics
- User engagement and playtime analysis
- Heatmaps and time-based user activity
- Filters by date, age group, game, and device language

## 🔍 Key Findings
- Total active users grew steadily from 47 (March) to 155 (November) in 2022
- Battle Pass rate peaked at 76.92% in April, dropping to 50% by December
- Age groups 10-19 showed the highest playtime across all quarters
- Average session time peaked in June (78:35 sec/user) and September (74:39 sec/user)
- English, Russian and Ukrainian language users dominated playtime metrics
